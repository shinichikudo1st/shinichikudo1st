## About

I'm a full stack developer with commercial experience building scalable backend systems, REST APIs, automation pipelines, and AI-integrated workflows for production teams.

My strongest work is in **Go**, **TypeScript**, **PostgreSQL/Supabase**, **Docker**, **Linux infrastructure**, and backend services that need to be reliable, observable, and maintainable. I also build frontend interfaces with **Next.js** and **React** when the product needs an end-to-end implementation.

I care about simple architecture, readable code, strong error handling, practical testing, and shipping systems that are easy for other developers to understand and extend.
